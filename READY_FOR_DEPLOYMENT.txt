STEP 1: Prepare Code

TROUBLESHOOTING QUICK REFERENCE
═══════════════════════════════════════════════════════════════════════════════

"Cannot connect to MongoDB"
   → Check MongoDB Atlas connection string
   → Verify username/password correct
   → Add IP whitelist: Network Access → 0.0.0.0/0

"API not responding from frontend"
   → Check backend URL in frontend/src/services/api.js
   → Rebuild frontend: npm run build
   → Check Heroku logs: heroku logs --tail

"Frontend shows blank page"
   → Check publish directory is "dist" in Netlify
   → Verify build command: npm run build
   → Check browser console for errors

"Import errors in development"
   → Expected - dependencies not in workspace
   → Will resolve automatically on deployment
   → Run locally: pip install -r requirements.txt
